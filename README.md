# FileManager
Đây là một chương trình quản lý file đơn giản!!

### requirements

- **file manager dependencies**  
  cài đặt: `pip install -r requirements.txt`

- **gui framework (nếu không dùng tkinter built-in)**  
  `pyqt5==5.15.7`  
  `pyqt6==6.4.0`

- **file operations**  
  `pathlib2==2.3.7`

- **search functionality**  
  `whoosh==2.7.4`

- **compression support**  
  `py7zr==0.20.2`

- **configuration management**  
  `configparser==5.3.0`
